üîå REST API Endpoints

Base URL: http://localhost:8082

POST /api/users

Create a new user

curl -X POST http://localhost:8082/api/users \
  -H "Content-Type: application/json" \
  -d '{
    "name": "John Doe",
    "email": "john@example.com",
    "password": "secret123"
  }'
GET /api/users

Get all users

curl http://localhost:8082/api/users
GET /api/users/{id}

Get user by ID

curl http://localhost:8082/api/users/1
GET /api/users?name={name}

Search users by name

curl http://localhost:8082/api/users?name=John
PUT /api/users/{id}

Update user

curl -X PUT http://localhost:8082/api/users/1 \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Jane Doe",
    "email": "jane@example.com",
    "password": "newpassword"
  }'
DELETE /api/users/{id}

Delete user

curl -X DELETE http://localhost:8082/api/users/1

üìñ Key Concepts Demonstrated

  • Dependency Injection: Constructor-based injection without Lombok
  • REST Controllers: @RestController, @RequestMapping, HTTP methods
  • Spring Data JPA: Repositories, custom queries, derived queries
  • DTOs & Records: UserResponse as Java record for immutability
  • Exception Handling: @RestControllerAdvice for global error handling
  • Bean Validation: @Valid, @NotBlank, @Email annotations
  • Configuration: @ConfigurationProperties for type-safe config
  • Modern Java: Records, pattern matching, .toList()
